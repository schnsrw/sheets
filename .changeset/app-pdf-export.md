Add **Download as PDF** (File menu). Exports the active sheet's used range as a real, searchable vector PDF (jsPDF + jspdf-autotable) — paginated across A4 pages, with column letters, row numbers, and a workbook/sheet title. Closes a MUST-have gap vs OnlyOffice + Google Sheets (we previously only had Print → "Save as PDF").
